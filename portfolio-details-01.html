The 'TestGuru' app

This is a web application on Ruby on Rails for creating and passing various tests.

  • Implementation of two types of users: administrator and regular user
  • SQLite 3 database for data storage
  • Authentication in the application via the Device gem
  • Localization via the I18n gem
  • Usage of Bootstrap version 5.3 for interface styling
  • Possibility to create GitHub Gists from questions
